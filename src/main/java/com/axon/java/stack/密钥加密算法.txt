
对称加密（AES、DES）
	•	特点：
	•	加密解密速度快。
	•	加密和解密使用相同的密钥。
	•	适合 大量数据加密。
	•	记忆提示：**“同”一个密钥，“快”速加密解密，适合“大”**量数据。

非对称加密（RSA、DSA）
	•	特点：
	•	加密和解密使用不同的密钥（公钥和私钥）。
	•	公钥加密，私钥解密。
	•	适用于 数据的安全传输、数字签名。
	•	记忆提示：**“不”同密钥，“公”钥加密，“私”钥解密，适合“安”**全通信。

哈希加密（MD5）
	•	特点：
	•	快速生成数据摘要。
	•	不可逆，常用于 数据完整性校验。
	•	适用于 密码存储，确保原始密码不可泄露。
	•	记忆提示：**“无”法逆向，“快”速校验，适合“密”**码存储。
